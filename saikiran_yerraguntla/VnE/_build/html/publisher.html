config = beehive.ClientConfig(

The Config function initalizes your config paramters, allowing you to: Connect to beehive-dev's rabbitmq server, Connect and transmit data to the respective virtual node, and Uses the respective authentication certificates for a secure connection.

                        
client = beehive.PluginClient(
name='(name_of_queue)',
config=config)

This Client function allows you to create your client script and connection. It allows you to declare a queue to send all your information and sets your configuration parameters to your config file.

The remaining part of the script takes in your software/program/data to be sent to beehive-dev.

At the end of your program, in order to send your resepctive data/plugin to rabbitmq server, you must add the following line of code:

                        
client.publish('(key)', (value))

By assigning the aggregate of data to a particular variable, it becomes the value you send to rabbitmq. Thus, the following piece of code defines a unique key it assigns to your data, that when published, the rabbitmq server will easily be able to identify and process it accordingly.
